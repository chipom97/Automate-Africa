Overview

The WhatsApp Odoo Meta Base module offers businesses a way to streamline communication processes. By seamlessly integrating with WhatsApp graph api, this module allows users to communicate efficiently and dynamically with multiple contacts by sending different types of content. Among its primary features are the ability to send template messages and session messages, which enable users to convey important messages, or information updates. The module also supports multimedia file transmission, so that users can send messages from particular documents in addition to text messages.

Features

  • Template Messaging
  • Broadcasting
  • Send/Receive Session Messages
  • Attachment Sharing and Receiving
  • Document-Based Messaging
  • Template Updates from Meta to Odoo
  • Dynamic WhatsApp Messaging Activation with access groups
  • Screenshots
